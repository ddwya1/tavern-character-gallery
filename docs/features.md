# 功能说明

## 封面墙

角色卡会先以封面形式展示。导入本地文件后不会立刻写进 SillyTavern，而是进入待导入状态。确认内容没问题，再从详情页执行导入。

## 角色卡解析

当前支持：

- PNG
- WEBP
- JSON

PNG 会读取 `tEXt` 块里的 `chara`、`ccv3`、`data`。  
JSON 会优先读取 `data` 字段，没有 `data` 时读取根对象。

## 详情整理

详情页会把角色卡内容拆成几个区域：

- 角色摘要
- 身份设定
- 行为与对话
- 提示词
- 开场白
- 世界书
- 正则
- 版本历史

每块都尽量按原始卡里的结构展示，不额外编造内容。

## 导入酒馆

导入时写入已连接 SillyTavern 的 `characters` 目录。  
如果文件重名，会自动追加序号，避免覆盖已有角色卡。

## 已禁用功能

聊天记录功能暂时不开放。相关入口保留为禁用状态，避免误点。
